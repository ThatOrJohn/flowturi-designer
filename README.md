# Flowturi Designer

A visual layout editor for [Flowturi](https://github.com/ThatOrJohn/flowturi) Sankey diagrams.

## Features

- Interactive node/link editing
- Optional property assignment

## Getting Started

```bash
npm install
npm run dev
```
